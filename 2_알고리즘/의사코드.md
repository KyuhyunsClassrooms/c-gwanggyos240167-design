# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
시작
1. 랜덤 시드 초기화 
2. 반복 (사용자가 '종료'를 선택할 때까지):
    - 가위(0), 바위(1), 보(2)를 선택하라는 메시지 출력
    - 사용자 선택 입력 (0, 1, 2 중 하나)
    - 만약 선택이 0, 1, 2 중 하나가 아니면:
        - 잘못된 입력 메시지 출력하고 종료
    - 컴퓨터 선택 랜덤으로 생성 (rand() % 3)
    - 사용자 선택과 컴퓨터 선택을 출력
    - 게임 결과 출력:
        - 비겼을 경우: "비겼습니다!"
        - 사용자가 이겼을 경우: "당신이 이겼습니다!"
        - 컴퓨터가 이겼을 경우: "컴퓨터가 이겼습니다!"
    - 게임을 계속할지 여부 묻기 (계속 할지 종료할지)
3. 종료

## 2. 핵심 기능 1: 사용자 입력 받기 및 유효성 검사
```
(핵심 기능 1의 논리적 절차를 작성하세요.)

사용자가 '가위(0)', '바위(1)', '보(2)' 중 하나를 입력하게 한다.
1. 사용자 입력 받기
    - 입력 받은 값이 0, 1, 2 중 하나인지 확인
2. 만약 입력이 0, 1, 2가 아니면:
    - "잘못된 입력" 메시지 출력
    - 프로그램 종료
3. 입력이 유효하면:
    - 해당 값을 `userChoice`에 저장하고 진행
```

## 3. 핵심 기능 2: 게임 결과 판단
```
사용자와 컴퓨터의 선택 비교:
    - 만약 `userChoice == computerChoice`이면:
        - "비겼습니다!" 출력
    - 만약 `(userChoice == 0 && computerChoice == 2) || (userChoice == 1 && computerChoice == 0) || (userChoice == 2 && computerChoice == 1)`이면:
        - "당신이 이겼습니다!" 출력
    - 나머지 경우:
        - "컴퓨터가 이겼습니다!" 출력
```
