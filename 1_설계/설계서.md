# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
(예: 학생 성적 관리 프로그램)
간단한 가위 바위 보 게임
## 2. 핵심 기능 목록
(구현할 기능들을 순서대로 나열하세요.)
1.  사용자 입력 받기
2.  컴퓨터의 선택 (랜덤)
3.  게임 결과 판단
4.  결과 출력

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 1: 배열 사용 (기본)**
```c
const char *choices[] = {"가위", "바위", "보"};
가위, 바위, 보를 숫자 인덱스 (0, 1, 2)로 구분하고, 그 숫자에 맞는 문자열을 출력  
```

**방법 2: 구조체 사용 (선택)**
```c
// 예시: 학생 정보를 구조체로 묶어서 관리
struct Student {
    char name[20];
    int kor;
    int eng;
    int math;
    float avg;
};
struct Student students[5];  // 학생 5명 저장
```
